Pydle

Pydle is a Wordle implementation with the newly released Pyscript framework from Anaconda Inc. First generate a new word, then fill out a guess and submit with the green button. A specific word length can be specified or a ? for a random word length. NOTE: Pyscript may require a few seconds to load as it is in an alpha state currently.
